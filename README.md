## 📌 Overview

This project was developed during my internship at **Larsen & Toubro (L&T)** to automate the extraction of structured information from PDF documents and convert it into well-formatted Excel spreadsheets.

The application significantly reduces manual data entry, improves processing accuracy, and streamlines document processing workflows.

> **⚠️ Note:**  
> The source code is **not included** in this repository due to internship confidentiality and intellectual property restrictions.
